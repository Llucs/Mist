Mist (WIP) — Weather. Simplified.

Minimalist weather app for Android with Material 3 Expressive design and
home-screen widget. Built with Kotlin, Jetpack Compose, and Glance.

  Features

- Home-screen widget — glanceable weather at a glance, no lag
- 7-day forecast with highs, lows, and precipitation chance
- Current weather with feels-like, humidity, wind, and UV index
- Search any city worldwide (powered by Open-Meteo, no API key needed)
- Auto-location via GPS
- Material 3 Expressive theme — vibrant, dynamic, smooth
- No ads, no sign-up, no tracking
- Open-source

  Tech Stack

- Kotlin + Jetpack Compose
- Material 3 (Expressive color scheme)
- Glance (widget framework)
- Open-Meteo API
- Coil (image loading)
- DataStore (preferences)
- Retrofit + OkHttp

  Build

```bash
git clone https://github.com/Llucs/Mist.git
cd Mist
./gradlew assembleDebug
```
